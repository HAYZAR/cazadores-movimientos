# 🏃 Cazadores de Movimientos

Actividad de análisis gráfico de movimiento para Física. 10 estaciones con 2 problemas cada una.

## Archivos

```
index.html        ← sitio principal
qr.html           ← generador de códigos QR para imprimir
images/           ← gráficas de los problemas (20 imágenes)
```

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub (ej. `cazadores-movimientos`)
2. Sube **todos** los archivos y la carpeta `images/`
3. Ve a **Settings → Pages → Branch: main → Save**
4. Tu URL será: `https://TU-USUARIO.github.io/cazadores-movimientos`

## Generar e imprimir los QR

1. Abre `https://TU-USUARIO.github.io/cazadores-movimientos/qr.html`
2. Verifica que la URL base sea correcta
3. Presiona **Ctrl + P** e imprime los 10 códigos QR
4. Recorta y pega uno en cada estación

## Calificación

- Cada problema resuelto correctamente = 1 punto
- Nota proporcional al total resuelto
- Premio (merienda) a los 3 mejores si superan 10 ejercicios resueltos
